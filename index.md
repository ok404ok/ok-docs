## 1. Introduction

OK is a community-first digital experiment built around a simple idea:

**Meaning is created by participation.**

OK does not promise innovation through complexity.  
It explores how shared attention, coordination, and ownership can transform something minimal into something valuable.

This document outlines the current structure, intent, and direction of OK.

---

## 2. What is OK?

OK is a social-native token designed to grow alongside its community.

- No hype-driven roadmap
- No artificial urgency
- No hidden mechanics

Every OK exists because someone chose to participate.

---

## 3. Core Principle

> Supply grows with the community.

The OK supply is directly tied to the number of followers.

- Each follower represents **100 OK**
- This mechanism aligns growth with real interest
- Empty growth = no advantage

The community itself defines the scale.
